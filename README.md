# Simple Crypto Staking App

This is a simple blockchain project for beginners.

This project helps you:
- Connect a crypto wallet
- Stake tokens
- Unstake tokens
- View staked balance
- Learn how staking works

This project is good for:
- Web3 learners
- Blockchain students
- Crypto developers who want practice

---

## What You Need

- Web browser
- MetaMask wallet
- Basic JavaScript knowledge
- A deployed staking smart contract

---

## How To Run

1. Download this repository
2. Open the folder
3. Open `index.html` in your browser
4. Connect your wallet
5. Enter staking contract address
6. Stake or unstake tokens

---

## Project Files

- `index.html` = Main page  
- `app.js` = Main logic  
- `config.js` = Settings  
- `wallet.js` = Wallet connect  
- `staking.js` = Staking logic  
- `ui.js` = Screen updates  
- `abi.js` = Contract ABI  
- `utils.js` = Helpers  
- `style.css` = Styles  
- `logger.js` = Logs
